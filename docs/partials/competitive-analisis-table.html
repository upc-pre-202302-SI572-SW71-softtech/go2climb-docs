PlantGuard Novel IoT-Based Plant Monitoring System (Absar et al., 2023)
Perfil Overview Sistema IoT con aplicación móvil que permite a los usuarios monitorear el estado de sus plantas. Provee acceso remoto para revisar desde cualquier lugar la humedad del suelo, temperatura y más, incluyendo un sistema de análisis de patrones para encontrar el mejor momento para regar la planta.

Un sistema inteligente, basado en IoT que proporciona a los usuarios de forma remota información relacionada con la temperatura, humedad e intensidad de humedad del suelo para el monitoreo de las condiciones de las plantas.

Es importante mencionar que el sistema se presentó como parte de una conferencia de ingeniería, por lo tanto no es una aplicación comercial.

Ventaja competitiva El usuario puede recibir recomendaciones que se adapten a sus patrones de tiempo, teniendo en consideración el máximo bienestar de la planta con el tiempo disponible del usuario. Presenta la información de varios sensores de manera sencilla.
Perfil de marketing Mercado objetivo Jardineros aficionados Agricultores a gran escala
Estrategias de marketing Publicidad directa N/A
Perfil de producto Productos y servicios
  • Dispositivo IoT con múltiples sensores, conectado por WiFi y sincronizado con la aplicación móvil, que permite recopilar datos como la humedad del suelo y temperatura.
  • Aplicación móvil con acceso a los datos sincronizados del dispositivo IoT que permite al usuario configurar el dispositivo, ver el historial de datos y obtener recomendaciones para el mejor cuidado de sus plantas.
Dispositivo IoT con sensores como el DHT-11 y de humedad del suelo, junto con una modulo WiFi ESP-8266 que le permite conectarse a la plataforma IoT de Blynk para monitorear diversos parámetros de la planta remotamente.
Precios y costos La aplicación móvil no tendría costo, pero se debe comprar el dispositivo IoT y vincularlo a una cuenta con una suscripción opcional para funcionalidades avanzadas. N/A
Canales de distribución Play Store y App Store para la aplicación móvil y a través de proveedores como Amazon para la venta del dispositivo IoT. N/A
